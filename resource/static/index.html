dxFoil

The dxFoil is a tool to create a foam core template for hot wire cutting.
Output file is a DXF format. Edit them with your favorit CAD.


Dxfoilはラジコンの主翼などのフォームコアを熱戦カットするときのテンプレートの元を作るためのツールです。
翼型データのDATファイルをアップロードしたり指定して、各種パラメータをしていすると、 DXF形式で作られます。適当なCADファイルで開いて好みの形に仕上げて使ってください。
